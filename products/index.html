PRODUCTS製品情報

Mold 金型

ミネベアミツミ本体のリソースも活用し、幅広い分野、サイズの金型に対応しています。
ギヤ・丸物はもちろん、形状物、外観部品なども対応いたします。サイズは金型重量1.5t程度まで対応可能。
ミネベアミツミ製の金型内圧センサー、温度センサーを組み込むことで、金型内部を可視化し、
製品のトレーサビリティーや金型の早期立ち上げに有効な金型も多数実績がございます。
また、流動解析を行い、適正な位置へのゲート配置のご提案、変形、ソリ解析などを事前に行う事で、
スムーズな金型立ち上げを実現いたします。

納期は最短2週間～、競争力のある価格でお客様のご要望にお応えしてまいります。

  • 2方向斜めスライド型

    斜め方向へのスライド型も多数実績あり。スライドストロークの長いものはシリンダーを用いて動作させるものもございます。

  • 3方向スライド型

    社内製のモールドベースにより多方向のスライドでも安定した精度を保証します。

  • WORM金型

    モーターや型開き動作を利用した動力で駒を回転させて製品を抜いています。

Sampleサンプル

対応成形材料：エンプラ、スーパーエンプラ（PPS、LCP、PEEK、PAI、PEIなど）に対応、熱硬化樹脂にも対応いたします。
製品公差は最小シングルµ～、ギヤ精度は旧JIS0級の実績もあります。金型加工精度は3µ～の保証が可能です。

  • クラウニングギヤ

    特許取得の特殊歯形をもち、静音性、低振動、効率改善に効果が有ります。音の問題でお困りでしたら是非ご相談ください。

  • WORM ギヤ

    駒を回転させて抜く歯車です。４形歯形のWORMもご相談ください。

  • 円弧ウォームホイール

    WORMギヤに嚙合わせる大きなアンダーカットを持った歯車です。静音性、噛合い率向上、効率改善に効果が有ります。

  • 車載
    シート部品

    丸い物や、ギヤだけではありません。このようなスライド、押切が多数ある形状部品も多くの実績を残しています。

  • ターミナルインサート
    インシュレーター

    インシュレーターは多数個取りでも安定した量産を可能にした金型づくりが得意です。薄いものはt=0.2㎜まで実績あり。

Mold Baseモールドベース

  • MOLD BASE

    日本国内生産の高精度モールドベースも販売しております。高品質、長寿命な金型には子品質モールドベースが不可欠です。

Mold Baseモールドベース

MOLD BASE

こだわりの金型はこだわりのモールドベースから。日本国内生産の高精度モールドベースを販売しております。ガンドリルも保有し、社内一貫生産体制で作られるモールドベースは様々な仕様に対応し、全加工でお届けします。超高精度なモールドベースをベースメーカーに引けを取らない価格でご提供。納期は2週間～

Processing Technology加工技術

  • ギヤ駒ミガキ

    精密加工されたギヤ歯面を精度を落とさずに磨き上げる技術が有ります。美しく磨きこまれたギヤは静かな駆動をもたらします。

  • クラウニングギヤ

    特殊な歯形を持つギヤです。歯形、製法で特許を取得。あらゆるアイテムで静音化が求められる中、ギヤの静音化に抜群の効果を発揮します。 低振動、効率改善にも効果が有ります。
